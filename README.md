# homebrew-lloyd

Homebrew tap for [Lloyd](https://github.com/benwbooth/lloyd) — a Bartender-style
menu-bar manager for notched Macs.

```sh
brew tap benwbooth/lloyd
brew trust benwbooth/lloyd   # newer Homebrew requires trusting third-party taps
brew install --cask benwbooth/lloyd/lloyd
```

The cask is updated automatically by CI on each tagged release of the main repo.
